\section{Impact of Reward Functions}

In this part of the homework, you will get hands-on experience with designing reward functions. 
Since iterating on reward functions by running RL on them can be slow, we will be using a faster model-based planner to get behavior from the defined 
reward functions almost instantaneously. The planner does this by leveraging the simulator itself to optimize for behavior with respect to the reward function.
